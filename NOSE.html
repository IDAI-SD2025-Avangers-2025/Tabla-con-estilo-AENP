Herramientas para el Desarrollo de Software

Los lenguajes de programación son programas destinados a la construcción de otros programas informáticos, dicho lenguaje está compuesto por símbolos y reglas sintácticas y semánticas, expresadas en forma de instrucciones y relaciones lógicas, mediante las cuales se construye el código fuente de una aplicación o pieza de software determinado.

lenguaje diseñado para definir y mejorar

muchas funcionalidades a las páginas web haciéndolas
más interactivas, útiles y atractivas para el usuario. 		Python es un lenguaje de programación informático
que se utiliza a menudo para crear sitios web y software,
automatizar tareas y realizar análisis de datos.
